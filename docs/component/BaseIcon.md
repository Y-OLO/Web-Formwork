# BaseIcon 字体图标

## # 功能
集成Vue-Awesome的字体图标 [去看看传送门](http://fontawesome.io/icons/)

## # 使用

```html
<template>
  <BaseIcon type="flag"></BaseIcon>
</template>
<script>
export default {}
</script>
```

## # API

BaseIcon props

参数 | 说明 | 类型 | 可选值 | 默认值
--- |---   |---  |---    |--- 
name | 字体图标样式 | String | — |  
size | 图标的大小，单位是 px | Number | String | — 
color | 图标的颜色 | String | — | #eee 

目前支持的图标有

- home
- bar-chart-o
- area-chart
- group
- cubes
- line-chart
- pie-chart
- user-plus
- dollar
- ellipsis-h

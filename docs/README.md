# <span id="asa">ASA Web Formwork<span>

> 架构的目的是管理复杂度，将复杂问题分而治之、有效管理。

## 下载

```bash
git clone http://git.asa/DoMoreDoBetter/ASA-Web-Formwork.git
```

## 安装依赖

推荐使用npm安装依赖

```bash
npm install
```

## 启动

```bash
npm run dev
```

## 单元测试

```bash
npm run test
```

## 构建

```bash
npm run build
```
